Color Scheme Analysis
Primary Color: #00ffff (Cyan/Neon Cyan)

Used as the main interactive color throughout the app
Applied to active buttons, selected states, glows, and highlights
Variable name: --color-neon-cyan
Secondary/Accent Colors:

Neon Pink: #ff00ff - Used for recording indicators, certain text highlights, and alternating glow effects
Variable: --color-neon-pink
Neon Purple: #9d00ff - Used for borders, sliders, and UI separators
Variable: --color-neon-purple
Neon Mint/Green: #39ff14 - Used for online status indicators and positive states
Variable: --color-neon-mint
Background Color: #000000 (Pure Black)

Variable: --color-bg
The app uses a cyberpunk aesthetic with a black background featuring a cyan grid pattern overlay
Text Color: #e0e0e0 (Light Gray)

Primary text color for body content
Additional text colors include:
White (#ffffff) for headers and emphasis
Gray shades (#gray-400, #gray-600) for secondary text
The cyan color is also used for glowing text effects
The app follows a cyberpunk/neon aesthetic with high contrast colors against the black background, creating that "CyberSec Command" surveillance system look.
